== Description ==


MediaPress myCRED addon allows site owners to give points to their users for uploading media. Different points can be given for different MediaPress actions.

Currently supported actions are:

*   Photo Upload
*   Video Upload
*	Audio Upload
*	Document Upload
*	Photo Delete
*	Video Delete
*	Document Delete


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `mpp-mycred.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
